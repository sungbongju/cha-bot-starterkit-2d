2D 이미지 아바타 만드는 법 (VRoid 없이!)
=========================================

VRoid Studio 로 만들기 어려운 로봇·동물·일러스트 마스코트는
이 2D 이미지 아바타 모드를 쓰면 됩니다. 별도 프로그램이 필요 없어요.
가지고 있는 캐릭터 PNG 그림만 이 폴더(public/avatar2d/)에 넣으면 끝!

1) 켜는 법
   이 레포(2D 에디션)는 2D 이미지 아바타가 "기본값"입니다.
   환경변수 설정 없이, 아래 이미지만 넣고 푸시하면 바로 동작해요.
   (사람형 VRoid 3D 로 바꾸려면 Vercel env VITE_AVATAR_KIND=vrm 추가 후 Redeploy.)

2) 넣을 이미지 (PNG, 배경 투명 권장)
   - idle.png   (필수) 기본/대기 표정 (입 닫은 모습)
   - talk.png   (선택) 입 벌린 표정. 봇이 말할 때 이 그림으로 바뀌어요.
   - wink.png   (선택) 윙크 표정. 인사하거나 캐릭터를 클릭하면 잠깐 나와요.

   idle.png 한 장만 넣어도 동작합니다(말할 때 음량에 맞춰 살짝 들썩여요).
   talk.png 를 같이 넣으면 진짜 입이 움직이는 것처럼 보입니다.

3) 그림 준비 팁
   - 배경을 투명하게 지우려면 remove.bg 같은 무료 웹사이트에 올리면 됩니다.
   - 정사각형~세로형, 한 변 600px 이상이면 선명해요.
   - idle / talk / wink 는 캐릭터 위치·크기를 똑같이 맞춰야 자연스럽게 바뀝니다.

동작 원리(궁금하면): 봇이 말할 때 TTS 음성의 음량을 실시간으로 분석해서,
소리가 크면 talk.png(입 벌림), 작으면 idle.png 로 번갈아 보여 줍니다.
모두 브라우저 안에서 처리되므로 외부 비용이 0원입니다.
