왜 PNG 방식인가?

아바타를 만드는 방법은 3가지가 있는데, PNG가 가장 쉽습니다.

방식난이도필요한 것비고
PNG 이미지 ⭐가장 쉬움 (5분)캐릭터 그림 1~3장로봇·동물·일러스트 무엇이든 가능
VRoid (3D)보통VRoid Studio로 캐릭터 제작사람형만 가능
Live2D어려움 (반나절~1일)Cubism 리깅 작업가장 부드럽지만 전문 작업
💡 로봇·동물·캐릭터 마스코트는 VRoid로 못 만들기 때문에, PNG 방식이 정답입니다.

준비물 체크리스트

  • ☐ 내 봇 프로젝트 폴더 (cha-bot-starterkit-2d 를 클론한 폴더)
  • ☐ 캐릭터 그림 — 배경이 투명한 PNG 1~3장 (만드는 법은 STEP 1에)
  • ☐ GitHub 계정 + 내 봇 레포 (이미 봇 만들 때 만든 것)

STEP 1. 캐릭터 그림 준비

1-1. 어떤 그림이 필요한가

최대 3장이지만, idle.png 한 장만 있어도 동작합니다. 더 생생하게 하려면 3장을 준비하세요.

파일명언제 보이나예시
idle.png (필수)평소 / 대기 중호기심(눈 동그란) 표정
talk.png (선택)봇이 말하는 동안웃는 표정
wink.png (선택)캐릭터를 클릭했을 때윙크 표정
💡 talk.png 가 없으면, 말할 때 idle 그림이 살짝 통통 튀는 것으로 대신합니다. 그래도 충분히 자연스러워요.

1-2. 그림이 아직 없다면

직접 그리거나, AI 이미지 생성(예: ChatGPT 이미지, Gemini 등)에게 캐릭터를 그려달라고 해서 받으세요. 표정만 살짝 다른 그림 2~3장을 받으면 idle/talk/wink 로 쓸 수 있습니다.

💡 표정을 따로 받기 어려우면, 한 장(idle.png)만으로 시작해도 됩니다. 나중에 talk/wink 를 추가하면 돼요.

1-3. 배경을 투명하게 만들기

그림 배경이 흰색/색깔이면 봇 화면에 네모난 배경이 같이 보입니다. 배경을 투명하게 지워야 깔끔합니다.

  1. remove.bg 접속 → https://www.remove.bg/ (무료)
  2. 내 캐릭터 그림 업로드
  3. 배경이 자동으로 지워짐 → Download 클릭 (투명 PNG로 저장됨)
✅ 저장된 파일을 그림판/뷰어로 열었을 때 배경이 체크무늬로 보이면 = 투명 처리 성공.

1-4. 3장의 크기·위치 맞추기 (중요)

idle / talk / wink 는 봇이 같은 자리에서 겹쳐서 바뀌기 때문에, 캐릭터의 크기와 위치가 같아야 표정이 바뀔 때 흔들리지 않습니다.

  • 가능하면 같은 원본에서 표정만 바꾼 그림을 쓰세요 (몸·위치 동일).
  • 이미지 크기는 세로 600px 이상이면 선명합니다.

1-5. 파일 이름 규칙 (실수 1순위!)

⚠️ 파일명을 정확히 — 모두 소문자, 확장자 .png:
idle.png   talk.png   wink.png
한글 이름(예: 호기심.png), 대문자(IDLE.PNG), 띄어쓰기 → 인식 안 됩니다.

STEP 2. 그림을 폴더에 넣기

2-1. 넣을 위치

내 봇 프로젝트 폴더 안의 public/avatar2d/ 폴더에 넣습니다.

my-bot/
  └─ public/
       └─ avatar2d/        ← 여기에 그림을 넣으세요
            ├─ idle.png
            ├─ talk.png     (선택)
            └─ wink.png     (선택)

avatar2d 폴더가 없으면 새로 만들면 됩니다 (이름 정확히 avatar2d).

2-2. 넣는 방법

윈도우 탐색기에서 my-bot\public\avatar2d\ 폴더를 열고, 준비한 PNG 파일들을 드래그해서 복사하면 됩니다.

STEP 3. 깃허브에 올리기

3-1. 명령어 (cmd / 터미널)

봇 프로젝트 폴더(my-bot)에서 아래 3줄을 입력하세요.

git add public/avatar2d
git commit -m "Add my avatar"
git push
명령뜻
git add새로 넣은 그림을 "올릴 목록"에 추가
git commit변경 사항을 기록(저장점 만들기)
git push내 GitHub 레포로 전송 → Vercel이 자동으로 다시 배포

3-2. (대안) 명령어가 어렵다면

GitHub 웹사이트에서 내 레포 → public/avatar2d 폴더 → Add file → Upload files 로 그림을 드래그 업로드해도 됩니다. (Commit changes 클릭)

STEP 4. 결과 확인

  1. push(또는 업로드) 후 1~2분 기다리면 Vercel이 자동 재배포합니다.
  2. 내 봇 주소를 열고 "대화 시작"
  3. 확인 포인트:
    • 봇이 말할 때 → 캐릭터 입(표정)이 움직임
    • 평소 → 둥둥 떠다님
    • 캐릭터 클릭 → 윙크 ✨
💡 이 레포(cha-bot-starterkit-2d)는 PNG 모드가 기본값이라 아바타 설정을 따로 안 해도 됩니다. 단, 채팅이 되려면 Vercel 환경변수에 본인 팀 번호 TEAM_ID 가 들어가 있어야 해요 (봇 만들 때 이미 설정한 값).

동작 원리 (궁금한 사람만)

봇이 말할 때 나오는 음성의 크기(볼륨)를 실시간으로 분석해서 — 소리가 크면 talk 표정, 작으면 idle 표정으로 부드럽게 바꾸고, 볼륨에 맞춰 살짝 통통 튀게 합니다. 모든 처리가 브라우저 안에서 일어나므로 추가 비용이 0원입니다.

이 복잡한 음성·애니메이션 코드는 스타터킷에 이미 다 들어 있습니다. 그래서 여러분은 그림만 넣으면 됩니다.

잘 안 될 때 (트러블슈팅)

증상원인 / 해결
아바타 자리에 "비어있어요" 안내가 뜸파일명·위치 확인. public/avatar2d/idle.png 가 소문자·정확한 이름인지.
캐릭터 뒤에 흰색/색깔 네모가 보임배경을 안 지움 → remove.bg 로 투명 PNG로 다시 저장.
말할 때 표정이 흔들리거나 어긋남idle/talk/wink 의 크기·위치가 다름 → 같은 크기·위치로 맞추기.
아바타는 뜨는데 채팅이 안 됨Vercel 환경변수 TEAM_ID(본인 팀 번호) 확인.
그림을 바꿨는데 안 바뀜push 했는지 + 1~2분 기다렸는지 확인. 브라우저 새로고침(Ctrl+F5).

정리

① 그림 준비(투명 PNG, 이름 idle/talk/wink) → ② public/avatar2d/ 에 넣기 → ③ git add · commit · push → ④ 1~2분 뒤 봇에서 확인. 끝!
