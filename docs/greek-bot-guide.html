그릭봇은 어떻게 동작하나

그릭봇은 세 부분으로 이루어져 있어요. 여러분이 직접 관리하는 건 ①지식과 ②아바타뿐이고, 어려운 코드는 스타터킷에 다 들어 있습니다.

[사용자 질문] │ ├─ 채팅 답변 ── 미들턴 서버의 팀 06 RAG (여러분이 채운 지식) + Gemma4 AI ├─ 음성 ────── STT(듣기) / TTS(말하기) ← 자동 └─ 아바타 ──── public/avatar2d/ 의 PNG 그림이 음성에 맞춰 입을 움직임
구성누가 관리?어디서?
① 지식(RAG)여러분middleton.p-e.kr/finbot/team/06/rag
② 아바타(그림)여러분레포 public/avatar2d/
③ 채팅/음성/렌더링 코드스타터킷(자동)건드릴 필요 없음
🔎 팀 번호 06 이 핵심 연결고리예요. 배포된 봇의 Vercel 환경변수 TEAM_ID=06 이 "이 봇은 06팀 지식을 쓴다"를 결정합니다.

STEP 1지식 채우기 (RAG)

봇이 똑똑해지려면 "그릭봇이 알아야 할 내용"을 넣어줘야 해요. 코딩 없이 웹페이지에서 텍스트만 붙여넣으면 AI가 자동으로 봇 지식으로 변환합니다.

1-1. RAG 관리 페이지 열기

https://middleton.p-e.kr/finbot/team/06/rag

1-2. 지식 추가 (가장 쉬운 방법)

  1. 알려주고 싶은 내용(옵션 개념 설명, 강의 노트, 자주 묻는 질문 등)을 그대로 붙여넣기
  2. 🤖 AI로 청크 만들기 클릭 → 30초~1분 후 자동으로 Q&A 형태의 "청크" 생성
  3. 미리보기 확인 → 저장
💡 JSONL 같은 형식 몰라도 됩니다. 그냥 글을 붙여넣고 버튼만 누르면 AI가 알아서 정리해요.

1-3. 잘 채우는 팁

  • 한 번에 너무 길게 말고, 주제별로 나눠서 여러 번 추가하면 검색 품질이 좋아요.
  • 예상 질문과 답을 같이 적어주면(예: "콜옵션이 뭐야? → ...") 봇이 더 잘 답해요.
  • 추가 후 봇에서 바로 물어보며 확인 → 부족하면 더 채우기.

STEP 2아바타 입히기 (PNG)

그릭봇 캐릭터 그림을 넣으면, 봇이 말할 때 입이 움직이고 평소엔 둥둥 떠다닙니다. 3D·리깅·코딩 전부 불필요 — 그림만 넣으면 돼요.

idle.png (기본)talk.png (말할 때)wink.png (클릭 시)

2-1. 그림 3장 준비

파일명언제 보이나
idle.png (필수)평소 / 대기 중
talk.png (선택)봇이 말하는 동안
wink.png (선택)캐릭터를 클릭했을 때
💡 idle.png 한 장만 넣어도 동작해요(말할 때 살짝 통통 튐). 3장 다 넣으면 더 생생합니다.

2-2. 배경 투명하게

배경이 있으면 봇 화면에 네모가 같이 보여요. remove.bg(무료)에 올려 배경을 지우고 PNG로 저장하세요. (뷰어에서 배경이 체크무늬면 성공)

2-3. 크기 맞추기 & 파일명

  • idle/talk/wink는 캐릭터 크기·위치를 같게 (표정 바뀔 때 안 흔들리게)
  • 세로 600px 이상 권장
⚠️ 파일명은 모두 소문자: idle.png talk.png wink.png. 한글·대문자·띄어쓰기 → 인식 안 됨.

2-4. 폴더에 넣기

my-bot/
  └─ public/
       └─ avatar2d/       ← 여기에 그림
            ├─ idle.png
            ├─ talk.png    (선택)
            └─ wink.png    (선택)

STEP 3올리고 확인하기

3-1. 깃허브에 올리기

봇 프로젝트 폴더에서:

git add public/avatar2d
git commit -m "Add Greek bot avatar"
git push

명령어가 어렵다면 GitHub 웹 → 레포 → public/avatar2d → Add file → Upload files 로 드래그 업로드해도 됩니다.

3-2. 자동 배포 & 확인

  1. push 후 1~2분 → Vercel 자동 재배포
  2. 봇 주소 열고 대화 시작
  3. 확인: 말할 때 입 움직임 / 평소 둥둥 / 클릭 윙크
🔎 채팅이 되려면 Vercel 환경변수 TEAM_ID=06 이 들어가 있어야 해요(봇 만들 때 설정한 값). 아바타는 이 레포가 PNG 기본이라 따로 설정 불필요.

잘 안 될 때

증상해결
아바타 "비어있어요"파일명·위치 확인 (public/avatar2d/idle.png, 소문자)
캐릭터 뒤 흰 네모배경 안 지움 → remove.bg로 투명 PNG
표정 바뀔 때 흔들림idle/talk/wink 크기·위치 맞추기

지식 추가했는데 답에 반영 안 됨RAG 페이지에서 저장됐는지 + 새 대화로 다시 질문
그림 바꿨는데 안 바뀜push + 1~2분 대기 + 브라우저 새로고침(Ctrl+F5)
